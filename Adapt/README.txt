This folder contains examples of an Adapted voice model. This model was created using 100 audio
files from a user with the codename "bn" and a database composed of over 6000 recordings.
The exmaples witch contain the word "in" in the filenames come from phrases that were used during the training of the voice model, the ones that contain the word "out", don't.

Esta carpeta contiene ejemplos de un modelo de voz Adapatado. Este modelo de voz se ha creado
utilizando 100 ficheros de audio de un usuario con el código identificativo "bn" y una
base de datos compuesta de más de 6000 grabaciones.
Los ejemplos que contienen la palabra "in" en sus nombres de fichero proceden de frases utilizadas durante el entrenamiento del modelo de voz, los que contienen la palabra "out", no.
